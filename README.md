This project contains directories corresponding to standalone configuration to run simulation or
real robot experimentations.
There is no dependencies between each directory.
This means that it is possible to give one to a person if you want them to be able to run a
particular experiment.

## Prerequisite

You need to install a recent version of docker compose by [following the instruction on the
docker documentation](https://docs.docker.com/compose/install/linux/).
If it is already installed, you can check that its version is greater or equal to `2.20` using the
command:
```
docker compose version
```
